# Claude-like decompositions and “generalizable” `m=3` cycles

This note records (and slightly formalizes) the “generalizable” definition and the associated theorem in
Knuth’s **“Claude’s Cycles”** (2026-02-28; rev 2026-03-02), p.4.

Implementation pointers:

- Lifting map + Hamiltonicity check: `claudescycles/generalize.py`
- Reproduction of paper counts (`11502`, `1012`, `996`, `4554`, `760`): `python -m claudescycles.knuth_m3 --out-dir artifacts/knuth_m3`

## Setup

Fix an odd integer `m >= 3`.

Let `G_m` be the directed graph on `Z_m^3` with arcs that “bump” one coordinate:

- bump `i`: `(i,j,k) -> (i+1,j,k)`
- bump `j`: `(i,j,k) -> (i,j+1,k)`
- bump `k`: `(i,j,k) -> (i,j,k+1)`

all modulo `m`.

For any `(I,J,K) ∈ Z_m^3`, define its fiber index

`S(I,J,K) := (I + J + K) mod m`.

### The hat map

Define `x^ ∈ Z_3` for `x ∈ Z_m` by

- `0^ = 0`
- `(m-1)^ = 2`
- `x^ = 1` for `0 < x < m-1`.

This is the `x^` map in Knuth’s note (p.4).

### Lifting a direction function from `m=3` to general odd `m`

Let `C` be a directed Hamiltonian cycle in `G_3`. Equivalently, `C` is given by a direction function

`d_C : Z_3^3 -> {0,1,2}`

where `d_C(i,j,k)=0` means “bump `i`”, etc.

Define the lifted direction function `d_C^(m) : Z_m^3 -> {0,1,2}` by the following rule.

Given `(I,J,K)`, compute

- `i := I^`, `j := J^`, `s := S(I,J,K)^`,
- `k := (s - i - j) mod 3`,

and then set

`d_C^(m)(I,J,K) := d_C(i,j,k)`.

This is exactly the construction Knuth uses to define “generalizable” cycles (p.4) and is implemented in
`claudescycles/generalize.py`.

### Definition (generalizable)

A Hamiltonian cycle `C` of `G_3` is **generalizable** if for every odd `m >= 3`, the lifted direction function
`d_C^(m)` generates a directed Hamiltonian cycle in `G_m`.

In practice the paper counts cycles that lift to Hamiltonian cycles for `m=5` and for both `m=5` and `m=7`.

### Definition (Claude-like decomposition)

Knuth defines a decomposition as **Claude-like** if it can be generated by a “small case-table” program: the
per-vertex permutation `d(I,J,K) ∈ S_3` depends only on whether each of `I`, `J`, and `S(I,J,K)` is equal to
`0`, equal to `m-1`, or in the interior `{1,2,...,m-2}`. (No other special cases are allowed.)

Equivalently, a Claude-like decomposition is determined by its restriction to the `3×3×3` case `m=3`, and the
general-`m` behavior is a lift under the hat map above.

## Theorem (Knuth 2026, p.4)

**Theorem.** A Claude-like decomposition is valid for all odd `m >= 3` if and only if each of the three
Hamiltonian cycles that it defines for `m=3` is generalizable.

### Proof

(*Only if*.) Suppose a Claude-like decomposition is valid for all odd `m >= 3`. In particular, for `m=3` it
induces three directed Hamiltonian cycles `C0, C1, C2` in `G_3`.

Because the decomposition is Claude-like, the direction functions for odd `m` are obtained from the `m=3`
direction functions by the hat-map lifting. Therefore, for each `c ∈ {0,1,2}` and each odd `m >= 3`, the
lifted direction function `d_{Cc}^(m)` is exactly the cycle-`c` direction function of the decomposition on `G_m`.

Since the decomposition is valid on `G_m`, that direction function must generate a Hamiltonian cycle in `G_m`.
Thus each `Cc` is generalizable.

(*If*.) Conversely, suppose the three `m=3` cycles `C0,C1,C2` are generalizable. Let `m >= 3` be odd.

By generalizability, each lifted direction function `d_{Cc}^(m)` generates a directed Hamiltonian cycle of
length `m^3` in `G_m`.

Furthermore, in a Claude-like decomposition the per-vertex triple `(d_{C0}^(m), d_{C1}^(m), d_{C2}^(m))` is a
permutation of `{0,1,2}` at every vertex, so the three cycles are arc-disjoint and their union covers all arcs.

Therefore the lifted cycles form a valid decomposition of `G_m` into three arc-disjoint directed Hamiltonian
cycles. ∎
